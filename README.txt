By Justin Teichman

This project simply records linux commands and stores them in a GUI.

Requirments:

- linux OS	
- c++ complier 
- qt5 libary

Steps to build and execute project:


	1. Download all files and place in a folder/directory.
	2. Execute: qmake-qt5
	3. Execute: make
	4. Execute: ./CommandGUI

Clear is not implemented as command but rather a function that clears the GUI text box.
